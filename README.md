## Engineering perspective

My MSc thesis, **“A Failure-Mode Analysis of an Agentic Coding System for Autonomous Software Engineering,”** examined not only whether coding agents solved tasks, but also how they reasoned, edited, tested, recovered, and failed across multi-step software engineering workflows.

That research strengthened the way I think about software quality, evaluation, debugging, and the difference between code that merely passes and engineering work that is genuinely clean and reliable.
